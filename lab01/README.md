# Lab 01 – Hello, World e Chamadas de Sistema

## Objetivo
Explorar chamadas de sistema (syscalls) no Linux, utilizando ferramentas como `strace` e o mecanismo vDSO, além de entender a diferença entre funções da libc e syscalls diretas.

## Conteúdo desta pasta
- `original` – códigos disponibilizados pelo professor para este laboratório
- `lab1.pdf` – enunciado original
- `lab1-pdesio.pdf` – PDF com as respostas preenchidas
- `README.md` – este arquivo

> **Nota:** Este laboratório não exige entrega de códigos. Os arquivos `.c` criados durante os experimentos são apenas para testes e não foram incluídos no repositório.
